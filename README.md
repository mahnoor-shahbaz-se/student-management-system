🎓 Student Management & Grading System (Java)

A console-based Java application that dynamically manages student records, calculates academic performance, and generates grade-based fee breakdowns using core programming fundamentals.

📌 Project Overview

This project simulates a basic academic management system where users can input multiple students and subjects. The system processes marks, calculates totals, averages, grades, GPA, and applies scholarship rules to determine semester fees.

It is built using Programming Fundamentals concepts only, focusing on logic building and clean modular design.

✨ Features
Dynamic input for number of students and subjects
Input validation for marks (0–100)
Automatic calculation of:
Total marks
Average marks
Letter grades
GPA conversion
Scholarship-based fee calculation
Class average computation
Structured output in tabular format
Modular function-based design
🧠 Concepts Used
Arrays (1D & 2D)
Loops (for, do-while)
Conditional statements
Functions (modular programming)
Input validation
Console formatting
📊 Grading System
Average Marks	Grade	GPA
85 - 100	A	4.0
75 - 84	B	3.0
65 - 74	C	2.0
50 - 64	D	1.0
Below 50	F	0.0
💰 Fee Structure (Scholarship Policy)
GPA Range	Discount	Final Fee
4.0	30%	70% of base fee
3.0+	20%	80% of base fee
2.0+	10%	90% of base fee
Below 2.0	0%	Full fee
